Página criada para o projeto da bateria eletrônica desenvolvida utilizando arduino.
Projeto "Arduino aplicado ao desenvolvimento de controladores MIDI de baixo custo para inclusão através da música", realizado para o curso de Bacharelado em Sistemas de Informação da Universidade Católica de Santa Catarina.

A seguir, os dados do template HTML5 utilizado para criação da página.
